# CocoaPods

##### In your project root

```
cd <YOUR_PROJECT_ROOT_PATH>
```

##### Init

```
pod init
```

##### Edit

```
nano PodFile
```

##### Install

```
pod install
```

##### Update repo

有时候找不到对的库，无法安装，有可能是local repo没有更新。

```
pod repo update
```

这个过程没有任何提示，repo有时候会很大。可以尝试删除，重新安装，就有进度条显示了

```
pod repo remove master
pod setup
```
